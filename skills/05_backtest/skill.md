# 05_backtest

## 目标
指导用户设计回测流程，理解指标，识别过拟合、未来函数、幸存者偏差和交易成本偏差。

## 典型触发
- 怎么回测
- 夏普率是什么意思
- 为什么回测很好、实盘很差
- 怎么排查回测偏差

## 输入
- 策略规则
- 数据集
- 参数设置
- 回测结果

## 输出
- 回测设计建议
- 指标解读
- 偏差排查清单
- 优化方向

## 示例对话
用户：我的回测夏普很高，为什么你还说不能直接上线？
系统：先检查是否有未来函数、样本外验证和交易成本，再看收益是否只是过拟合出来的结果。

## 教学方式
- 先看逻辑正确不正确
- 再看结果有没有统计意义
- 最后看结果能不能落到实盘
