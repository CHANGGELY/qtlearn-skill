# 16_progression

## 目标
建立用户成长体系，包括评分量表、学习档案和成长等级，追踪学习进度并提供进阶激励。

## 典型触发
- 我现在是什么水平
- 我进步了多少
- 我还需要学什么
- 我的学习档案

## 输入
- 用户学习历史
- 测验成绩
- 项目完成情况
- 策略设计质量

## 输出
- 当前等级与称号
- 能力雷达图
- 学习进度条
- 下一级解锁条件

## 成长等级体系

### L1 量化小白
**解锁条件**：完成 onboarding 诊断
**能力要求**：
- 了解量化基本概念
- 知道 A 股交易规则
- 能说出 3 个常用因子

### L2 策略学徒
**解锁条件**：完成第一个策略设计
**能力要求**：
- 能把想法写成规则
- 会基础数据清洗
- 能跑通简单回测

### L3 回测工程师
**解锁条件**：完成回测并识别出 3 个偏差
**能力要求**：
- 理解夏普、回撤、胜率
- 能排查未来函数
- 会做样本外验证

### L4 风控专员
**解锁条件**：设计完整风控体系
**能力要求**：
- 会仓位管理
- 会设置止损规则
- 能控制行业/市值暴露

### L5 策略研究员
**解锁条件**：完成因子工程或行业轮动项目
**能力要求**：
- 能挖掘和验证新因子
- 会做因子正交化
- 能设计多因子组合

### L6 量化高手
**解锁条件**：完成实盘准备检查表
**能力要求**：
- 理解实盘与回测差异
- 会监控和异常处理
- 有完整项目作品集

## 评分量表

### 概念理解（满分 100）
- 基础概念：20 分
- 交易规则：20 分
- 数据知识：20 分
- 回测原理：20 分
- 风控意识：20 分

### 实操能力（满分 100）
- 数据处理：25 分
- 策略编写：25 分
- 回测执行：25 分
- 结果分析：25 分

### 项目质量（满分 100）
- 逻辑完整性：30 分
- 代码规范性：20 分
- 回测严谨性：30 分
- 文档清晰度：20 分

## 学习档案模板
```
用户：[用户名]
当前等级：L3 回测工程师
总学习时长：XX 小时
完成任务：XX 个
通过测验：XX 次

能力雷达图：
- 概念理解：75/100
- 实操能力：60/100
- 项目质量：55/100

最近学习：
- 完成：05_backtest 回测专项
- 进行：06_risk_control 风控专项

下一目标：
- 完成风控体系设计
- 升级到 L4 风控专员
```

## 示例对话
用户：我现在是什么水平？

Skill：
1. 展示当前等级和称号
2. 显示能力雷达图
3. 列出已完成和进行中任务
4. 说明下一级解锁条件
5. 推荐下一步学习内容

## 教学方式
- 用等级和称号提供成就感
- 用雷达图可视化能力短板
- 用进度条提供目标感
- 定期回顾学习档案
