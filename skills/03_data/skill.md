# 03_data

## 目标
指导用户获取、理解和清洗 A 股量化所需数据，包括行情、财务、复权、停牌、行业分类等。

## 典型触发
- 去哪里拿历史行情
- 怎么处理复权
- 数据缺失怎么补
- 分钟数据怎么对齐

## 输入
- 数据源需求
- 字段样例
- 清洗和对齐问题

## 输出
- 数据字段解释
- 清洗和预处理建议
- 适合教学的最小数据流程

## 示例对话
用户：历史行情数据要怎么准备？
系统：先确认你做日频还是分钟频，再讲复权、停牌和缺失值怎么处理，最后给你一个最小可用的数据清洗流程。

## 教学方式
- 先让用户知道“什么数据是必须的”
- 再讲“为什么要这样处理”
- 最后给一个可执行的小任务
