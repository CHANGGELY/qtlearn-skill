# 15_strategy_cases

## 目标
接入项目中已有的真实策略案例，通过案例教学帮助用户理解策略设计的完整流程。

## 典型触发
- 给我看看真实的策略案例
- 小市值策略是怎么做的
- 基本面选股策略怎么设计
- 量价策略有什么套路

## 输入
- 用户想学的策略类型
- 用户当前水平
- 可用数据条件

## 输出
- 策略案例拆解
- 核心逻辑讲解
- 可复用的设计模式
- 针对用户水平的改造建议

## 项目策略库
可接入你项目中的以下策略案例：

### 小市值系列
- **小市值_基本面优化1**：市值 + ROE + 净利润率
- **小市值_基本面优化2**：市值 + 自由现金流 + 归母净利润同比增速
- **小市值_量价优化1**：市值 + 成交额Mean + 成交额Std
- **小市值_量价优化2**：市值 + 当前回撤 + 成交量特征

### 龙头行业策略
- **龙头行业小市值**：行业龙头 + 小市值精选
- **主板战士**：主板筛选 + 多因子打分

### 策略设计模式
每个案例包含：
1. **选股池定义**：全A/主板/特定行业
2. **筛选条件**：市值、财务、量价门槛
3. **打分因子**：ROE、净利润率、成交额等
4. **排序与限制**：取前N只，行业/个股上限
5. **调仓规则**：定期调仓 + 应急条件

## 示例对话
用户：我想学小市值策略，你能给我讲一个真实案例吗？

Skill：
1. 先确认你的数据条件（日频/分钟频、财务数据范围）
2. 给你看一个“小市值+基本面”案例
3. 拆解选股池、筛选条件、打分逻辑
4. 让你动手改一个参数，看回测变化
5. 引导你设计自己的版本

## 教学方式
- 用真实案例替代抽象概念
- 强调“为什么这样设计”
- 鼓励用户在案例基础上改造创新
