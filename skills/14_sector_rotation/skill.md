# 14_sector_rotation

## 目标
指导用户构建行业轮动模型，掌握行业景气度分析、动量轮动、资金流向、宏观映射等轮动策略。

## 典型触发
- 怎么判断哪个行业会涨
- 行业轮动怎么做
- 宏观经济怎么映射到行业
- 行业动量有效吗

## 输入
- 行业分类标准（申万/中信）
- 行业基本面数据
- 行业资金流向
- 宏观指标

## 输出
- 行业评分框架
- 轮动信号定义
- 行业配置权重
- 轮动频率与再平衡规则

## 示例对话
用户：我想做行业轮动，但不知道从哪里入手？

Skill：
1. 先选行业分类：申万一级还是自定义？
2. 再选轮动逻辑：动量、景气度、资金还是宏观？
3. 定义行业评分指标（如行业动量+盈利预期+资金流向）
4. 确定轮动频率（月/季度）
5. 给出回测验证方案

## 教学方式
- 区分行业轮动的不同逻辑（动量 vs 反转 vs 景气度）
- 强调行业轮动的交易成本
- 用信息比率和胜率评估轮动效果
