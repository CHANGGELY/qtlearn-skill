# A股量化培训大师Skill

一个面向零基础到进阶用户的 A 股量化学习与实战训练 Skill。

## 一、项目定位
A股量化培训大师Skill 不是普通问答工具，而是一个“会诊断、会教学、会测评、会路由、会项目化训练”的量化学习总 Skill。

它的目标不是只回答一个问题，而是把用户逐步培养成具备以下能力的人：
- 能理解 A 股量化的核心概念
- 能把模糊炒股想法变成量化策略
- 能获取和处理历史数据
- 能设计并解释回测
- 能理解风险控制与组合管理
- 能具备走向实盘前的工程认知

## 二、适合谁用
- 想系统学习 A 股量化的小白
- 已经会一点 Python，但不会系统做策略的人
- 有炒股经验，但不会把主观逻辑量化的人
- 想做量化项目、作品集、比赛展示的人

## 三、如何调用
### 1. 直接对话
你可以直接说你的问题，例如：
- “我完全不懂量化，从哪开始？”
- “帮我把这个选股想法变成策略”
- “为什么我的回测很好但实盘不行？”

### 2. 输入命令触发
你也可以用命令快速进入不同模式：
- `/qtlearn`：开始学习诊断
- `/qtpath`：生成个性化学习路径
- `/qtcheck`：测评、批改、检查理解
- `/qtstrategy`：进入策略设计专项
- `/qtbacktest`：进入回测专项
- `/qtrisk`：进入风控专项
- `/qtlive`：进入实盘专项
- `/qtproject`：启动项目制学习
- `/qtskill`：查看可用子 skill
- `/qtreview`：进入复盘与诊断
- `/qtfaq`：快速问答

### 3. 推荐起手式
最推荐的输入是：
- `/qtlearn 我想系统学A股量化，但我现在是零基础`

这样 Skill 会先判断你的基础、目标和时间投入，再给你最短学习路径。

## 四、操作步骤
1. 先说出你的当前水平或目标
2. Skill 自动判断你适合学什么
3. 它会把你路由到对应子 skill
4. 你按它给的小任务继续回答或实操
5. 它再检查、纠错，并给出下一步

## 五、最小 demo 流程
### Demo 1：零基础学习
用户输入：

## 六、失败示例（更像教练）
### 失败示例 1：用户问题太泛
用户：我想学量化。

Skill 应该追问：
- 你现在会 Python 吗？
- 你想先学概念还是实战？
- 你偏向选股、择时还是实盘？

### 失败示例 2：用户想直接下结论
用户：你直接告诉我什么股票会涨。

Skill 应该回应：
- 不能直接给确定性投资建议
- 但可以把你的选股逻辑改写成可回测的研究问题

### 失败示例 3：用户目标和时间不匹配
用户：我每周 3 小时，但想一周做完实盘系统。

Skill 应该说明：
- 目标需要拆阶段
- 先完成策略、回测和风控，再谈实盘

## 七、核心设计亮点
### 1. 总 Skill + 子 Skill 结构
总 Skill 负责诊断、编排、路由和追踪学习路径。
子 Skill 负责具体模块教学与任务执行。

### 2. 主动命令 + 被动语义触发
用户既可以输入命令，也可以直接自然对话。

### 3. 苏格拉底式教学
不是一口气灌输知识，而是：
- 先提问了解用户水平
- 再讲用户最可能感兴趣的部分
- 再布置一个最小练习
- 再检查理解并进阶

### 4. 项目制闭环
整个 Skill 强调闭环：
诊断 → 教学 → 练习 → 测评 → 纠错 → 项目 → 复盘

## 十、与现有量化项目的结合
这个 Skill 可以直接接入你现有的量化知识、策略和因子库：

### 因子库接入
- 市值、ROE、净利润率、成交额等基础因子
- 一级行业、风险标签等分类因子
- 当前回撤、成交额 Std 等衍生因子

### 策略案例接入
- 小市值系列：基本面优化、量价优化
- 龙头行业策略：行业龙头 + 小市值精选
- 主板战士：主板筛选 + 多因子打分

### 数据与回测接入
- 在 **策略设计** 阶段调用你已有的选股逻辑
- 在 **数据** 阶段复用你现成的数据处理经验
- 在 **回测** 阶段接入你已有的回测框架和指标口径
- 在 **风控** 阶段结合你现有仓位管理、风险标签和暴露控制思路
- 在 **项目制训练** 阶段用你真实的策略案例做教学样本

这样它不是空泛教学，而是能和你的真实项目互相喂养。

## 十一、成长等级体系（L1-L6）

| 等级 | 名称 | 解锁条件 | 核心能力 |
|------|------|----------|----------|
| L1 | 量化小白 | 完成 onboarding | 了解基础概念 |
| L2 | 策略学徒 | 完成第一个策略设计 | 能把想法写成规则 |
| L3 | 回测工程师 | 识别 3 个回测偏差 | 能排查未来函数 |
| L4 | 风控专员 | 设计完整风控体系 | 会仓位与止损管理 |
| L5 | 策略研究员 | 完成因子/轮动项目 | 能挖掘验证新因子 |
| L6 | 量化高手 | 完成实盘准备检查 | 有完整项目作品集 |

### 评分维度
- **概念理解**：基础概念、交易规则、数据知识、回测原理、风控意识
- **实操能力**：数据处理、策略编写、回测执行、结果分析
- **项目质量**：逻辑完整性、代码规范性、回测严谨性、文档清晰度

### 成就徽章
- 初出茅庐：完成 onboarding
- 策略新手：完成第一个策略设计
- 回测达人：完成回测专项
- 风控卫士：设计完整风控体系
- 因子猎人：完成因子工程项目
- 择时高手：完成择时模型
- 行业专家：完成行业轮动策略
- 案例大师：学习 5 个策略案例
- 六边形战士：所有能力维度 80+
- 量化高手：达到 L6 等级

### 3. 它兼顾知识、策略、工程和合规
不仅会讲概念，也会讲回测、风控、数据和实盘约束。
